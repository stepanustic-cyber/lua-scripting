5. Переменные и операторы

  • Переменные могут быть локальными и глобальными.
  • Условные операторы позволяют выполнять код в зависимости от истинности условия.
  • Циклы используются для многократного выполнения блока кода.

Если объявление переменной происходит с помощью ключевого слова local, переменная носит локальный характер и её область видимости ограничена текущим блоком. Без этого слова создаётся глобальная переменная. Третий вид переменных — поля таблиц.

Условные операторы в Lua начинаются с ключевого слова if, за которым следует условие и блок кода, выполняемый при истинности условия. Для создания циклов можно использовать while, repeat…until или числовой цикл for. При необходимости выражение break завершает исполнение цикла, пропуская оставшиеся команды. Ниже представлены обновленные примеры:

Конструкция Пример Описание
Объявление переменной local playerName = "Admin" Создание защищенной локальной переменной с именем игрока.
Условный оператор if health <= 0 then print("Game Over") end Проверка здоровья игрока: если оно 0 или меньше, игра окончена.
Цикл while while isLoading do showSpinner() end Пока идет загрузка (true), отображается индикатор на экране.
Цикл for for score = 10, 50, 10 do print("Бонус: " .. score) end Цикл прибавляет по 10 очков за шаг от 10 до 50.
