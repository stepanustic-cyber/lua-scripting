5.2. Операторы сравнения

В языке Lua существует всего 6 видов операторов сравнения:

  • == — равенство
  • ~= — неравенство
  • < — меньше
  • > — больше
  • <= — меньше или равно
  • >= — больше или равно

Пример кода:

local a = 10
local b = 20

print(a == b) -- Равенство: false
print(a ~= b) -- Неравенство: true (в Lua используется ~= вместо !=)
print(a < b) -- Меньше: true
print(a > b) -- Больше: false
print(a <= 10) -- Меньше или равно: true
print(a >= 15) -- Больше или равно: false

-- Сравнение строк (идет по алфавиту/кодам символов)
print("abc" == "abc") -- true
print("apple" < "banana") -- true
