3. Типы данных

В языке Lua представлены следующие типы данных:

Тип Описание Пример кода
nil Используется для удаления переменной или обозначения пустоты. local x = nil
number Любые числа: целые (integers) или дробные (floats). local price = 149.99
string Текст, заключенный в одинарные или двойные кавычки. local msg = "Lua Rocks!"
boolean Логический тип данных. local isOpen = false
table Единственная сложная структура данных (массивы и объекты). config = { "volume", 100 }
function Функции, которые можно хранить как обычные переменные. function add(a, b) return a+b end
userdata Специфические данные, созданные на языке C. local myFile = io.open("test.txt")
thread Сопрограммы (coroutines) для параллельной логики. local co = coroutine.create(myFunc)
