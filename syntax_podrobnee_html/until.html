🔄 Ключевое слово: until

Условие выхода из цикла repeat...until

until — это ключевое слово, которое завершает цикл repeat...until. Оно проверяет условие после выполнения тела цикла, поэтому цикл всегда выполняется хотя бы один раз. Цикл продолжается, пока условие ложно, и заканчивается, когда условие становится истинным.

📌 Синтаксис:
repeat
    -- тело цикла
until условие
🔄 Отличие от while:
• repeat...until — проверяет условие после выполнения (постусловие)
• while...end — проверяет условие до выполнения (предусловие)
• repeat...until всегда выполняется минимум 1 раз

Примеры

-- Простейший пример: счётчик local i = 1 repeat print("Итерация " .. i) i = i + 1 until i > 3 -- Вывод: -- Итерация 1 -- Итерация 2 -- Итерация 3
-- Цикл выполнится хотя бы один раз, даже если условие сразу истинно local x = 10 repeat print("Тело цикла выполнилось") until x > 5 --> Тело цикла выполнилось (один раз)
-- Ввод данных до получения корректного значения local userInput repeat print("Введите число от 1 до 10:") userInput = io.read("*n") until userInput >= 1 and userInput <= 10 print("Спасибо! Вы ввели " .. userInput)
-- Поиск элемента в таблице local fruits = { "яблоко", "банан", "апельсин", "груша" } local searchFor = "апельсин" local index = 1 local found = false repeat if fruits[index] == searchFor then found = true else index = index + 1 end until found or index > #fruits if found then print(searchFor .. " найден на позиции " .. index) else print(searchFor .. " не найден") end

Сравнение repeat...until и while

-- while (проверка ДО) local i = 1 while i > 5 do print("while: " .. i) -- не выполнится ни разу i = i + 1 end -- repeat...until (проверка ПОСЛЕ) local j = 1 repeat print("repeat: " .. j) -- выполнится 1 раз j = j + 1 until j > 5
-- Пример: запрос пароля (repeat удобнее) -- Вариант с repeat...until local password repeat print("Введите пароль:") password = io.read() until password == "secret123" -- Вариант с while (нужна дополнительная переменная) local password2 = "" while password2 ~= "secret123" do print("Введите пароль:") password2 = io.read() end

Важные особенности

⚠️ Область видимости переменных:
-- Переменные, объявленные внутри repeat, видны в условии until repeat local x = 10 -- локальная переменная print(x) until x > 5 -- ОШИБКА! x здесь недоступна -- Правильно: объявлять переменную ДО цикла local x repeat x = 10 print(x) until x > 5 -- ✅ работает
⚠️ Отсутствие end:
-- В отличие от других циклов, repeat...until НЕ требует end -- while требует end while condition do -- тело end -- ← нужен end -- repeat...until не требует repeat -- тело until condition -- ← нет end

Практические примеры

-- Игровой цикл (до получения команды выхода) local command repeat print("Введите команду (move, attack, exit):") command = io.read() if command == "move" then print("Вы переместились") elseif command == "attack" then print("Вы атаковали") elseif command == "exit" then print("Выход из игры") else print("Неизвестная команда") end until command == "exit"
-- Генерация случайного числа до угадывания math.randomseed(os.time()) local secret = math.random(1, 100) local guess local attempts = 0 repeat print("Угадайте число от 1 до 100:") guess = io.read("*n") attempts = attempts + 1 if guess < secret then print("Больше!") elseif guess > secret then print("Меньше!") end until guess == secret print("Поздравляю! Вы угадали за " .. attempts .. " попыток")
-- Обработка меню с валидацией local choice repeat print("===== МЕНЮ =====") print("1. Начать игру") print("2. Загрузить игру") print("3. Настройки") print("4. Выход") print("Выберите пункт:") choice = io.read("*n") if choice == 1 then print("Новая игра...") elseif choice == 2 then print("Загрузка...") elseif choice == 3 then print("Настройки...") elseif choice == 4 then print("До свидания!") else print("Неверный выбор! Попробуйте снова.") end until choice == 4
💡 Когда использовать repeat...until:
  • Ввод/вывод данных (запрос пароля, валидация ввода)
  • Игровые циклы (до получения команды выхода)
  • Меню и диалоги с пользователем
  • Любая ситуация, когда тело цикла должно выполниться хотя бы один раз

Частые ошибки

❌ Перепутан порядок (как в while):
-- Неправильно (while-синтаксис) repeat i > 10 until print(i) i = i + 1 end -- Правильно repeat print(i) i = i + 1 until i > 10
❌ Бесконечный цикл (условие никогда не становится истинным):
local i = 1 repeat print(i) -- забыли увеличить i until i > 5 --> бесконечный цикл!
❌ Использование break без необходимости:
-- repeat...until сам проверяет условие, break нужен редко repeat local input = io.read() if input == "quit" then break -- обычно лучше вынести в условие until end -- обработка input until false -- Лучше: repeat local input = io.read() if input ~= "quit" then -- обработка input end until input == "quit"
