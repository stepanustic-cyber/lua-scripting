❓ Ключевое слово: if

Условный оператор if

if — выполняет блок кода при истинности условия. В Lua false и nil — ложь, всё остальное (включая 0 и "") — истина.

📌 Синтаксис:
if условие then ... end
if ... then ... else ... end
if ... then ... elseif ... then ... else ... end

Примеры

-- Простой if local age = 18 if age >= 18 then print("Доступ разрешён") end
-- if-else local score = 65 if score >= 60 then print("Зачёт") else print("Незачёт") end
-- elseif (несколько условий) local mark = 85 if mark >= 90 then print("5") elseif mark >= 75 then print("4") elseif mark >= 60 then print("3") else print("2") end
-- Логические операторы and / or local user = { vip = true, amount = 1500 } if user.vip and user.amount > 1000 then print("Скидка 15%") elseif user.amount > 1000 then print("Скидка 10%") end

Важные особенности

⚠️ 0 и "" — это true!
if 0 then print("0 — истина") end if "" then print("пустая строка — истина") end
💡 Guard clause (ранний возврат):
function process(data) if not data then return nil, "нет данных" end if type(data) ~= "table" then return nil, "не таблица" end -- основная логика return data.value * 2 end

Частые ошибки

❌ Забыт then: if x > 5 then — обязательно!
❌ else if вместо elseif: в Lua пишется слитно — elseif
❌ Забыт end: каждый if требует закрывающего end
