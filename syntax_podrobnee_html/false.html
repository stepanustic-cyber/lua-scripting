Логическое значение «ложь» (false)

false — это одно из двух логических значений (boolean) в Lua. Вместе с true оно используется для управления потоком выполнения программы. В условных операторах false (наряду с nil) считается ложным значением.

📌 Важно:
• В Lua только false и nil являются ложными
• Все остальные значения (включая 0, "", {}) считаются истинными
• false — это значение, а nil — отсутствие значения

Примеры

-- Прямое использование false local isActive = false if isActive then print("Активно") else print("Неактивно") end --> Неактивно
-- false в логических операциях print(false and true) --> false print(false or true) --> true print(not false) --> true
-- Сравнение false и nil local a = false local b = nil if a == nil then print("false равен nil") else print("false не равен nil") end --> false не равен nil -- Оба ложные, но не равны друг другу if a == b then print("Равны") else print("Не равны") end --> Не равны
-- Проверка на false local function processValue(value) if value == false then print("Получено явное false") elseif value == nil then print("Получен nil") else print("Получено истинное значение: " .. tostring(value)) end end processValue(false) --> Получено явное false processValue(nil) --> Получен nil processValue(0) --> Получено истинное значение: 0

Важные особенности

⚠️ false не равен nil:
-- Оба ложные, но это разные значения local isEnabled = false local isSet = nil -- Разное поведение в or local result1 = isEnabled or "default" --> false (не "default"!) local result2 = isSet or "default" --> "default" print(result1, result2)
💡 Когда использовать false:
  • Явное указание «ложного» состояния (в отличие от «отсутствия» nil)
  • Возврат из функции, когда операция не удалась, но nil имеет другое значение
  • Флаги и переключатели
-- Функция, которая может вернуть false (осмысленное значение) local function userHasPermission(user, action) if not user then return nil -- пользователь не существует end -- проверка прав if user.role == "admin" then return true elseif user.role == "guest" then return false -- гость не имеет прав (осмысленный ответ) end end
