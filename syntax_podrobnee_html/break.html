⛔ Ключевое слово: break

Досрочный выход из цикла (break)

break — это оператор, который немедленно завершает выполнение ближайшего цикла (while, repeat...until или for). После break программа продолжает выполнение с первой инструкции после цикла.

📌 Важно:
• break работает только внутри циклов
• Завершает только самый внутренний цикл (не все сразу)
• Нельзя использовать вне цикла (будет ошибка)
• Обычно используется вместе с условием if

Примеры

-- Простейший пример: выход при достижении условия for i = 1, 10 do if i == 5 then break -- выход, когда i станет равно 5 end print(i) end -- Вывод: 1 2 3 4
-- break в цикле while local i = 1 while true do -- бесконечный цикл print("Итерация " .. i) if i >= 3 then break -- выход из бесконечного цикла end i = i + 1 end -- Вывод: Итерация 1, Итерация 2, Итерация 3
-- break в цикле repeat...until local i = 1 repeat print(i) if i == 3 then break end i = i + 1 until i > 5 -- Вывод: 1 2 3

Поиск элемента в таблице

-- Классический пример: поиск значения в таблице local names = { "Анна", "Иван", "Мария", "Пётр", "Ольга" } local searchName = "Мария" local foundIndex = nil for i = 1, #names do if names[i] == searchName then foundIndex = i break -- нашли — дальше искать смысла нет end end if foundIndex then print(searchName .. " найдено на позиции " .. foundIndex) else print(searchName .. " не найдено") end

break во вложенных циклах

-- break выходит ТОЛЬКО из внутреннего цикла for i = 1, 3 do print("Внешний цикл: " .. i) for j = 1, 5 do if j == 3 then break -- выход только из внутреннего цикла end print(" Внутренний цикл: " .. j) end print("Внешний цикл продолжается") end -- Вывод: -- Внешний цикл: 1 -- Внутренний цикл: 1 -- Внутренний цикл: 2 -- Внешний цикл продолжается -- Внешний цикл: 2 -- Внутренний цикл: 1 -- Внутренний цикл: 2 -- Внешний цикл продолжается -- Внешний цикл: 3 -- Внутренний цикл: 1 -- Внутренний цикл: 2 -- Внешний цикл продолжается
-- Выход из всех циклов (нужен флаг) local found = false for i = 1, 5 do for j = 1, 5 do if i * j == 12 then print("Найдено: " .. i .. " * " .. j .. " = 12") found = true break -- выход из внутреннего цикла end end if found then break -- выход из внешнего цикла end end

Важные особенности

⚠️ break нельзя использовать вне цикла:
function badExample() break -- ОШИБКА! break вне цикла end -- Правильно: break только внутри for, while, repeat function goodExample() for i = 1, 10 do if i > 5 then break -- ✅ можно end end end
⚠️ break в repeat...until:
local i = 1 repeat if i == 3 then break -- ✅ работает end print(i) i = i + 1 until i > 5 -- условие даже не проверяется, если break сработал
⚠️ break не принимает аргументов:
-- Нельзя выйти из нескольких циклов сразу break 2 -- ОШИБКА! В Lua нет break с числом -- Используйте флаг или return local function search() for i = 1, 10 do for j = 1, 10 do if i * j == 42 then return i, j -- выход сразу из функции end end end return nil end

break vs return

📊 Отличия:
  • break — выходит только из цикла, функция продолжает работу
  • return — выходит из функции целиком, возвращая значение
  • break нельзя использовать вне цикла
  • return можно использовать в любом месте функции
-- break vs return function exampleBreak() for i = 1, 5 do if i == 3 then break -- выходим из цикла end print(i) end print("Функция продолжается после цикла") -- ✅ выполнится end function exampleReturn() for i = 1, 5 do if i == 3 then return -- выход из функции end print(i) end print("Эта строка НЕ выполнится") end exampleBreak() --> 1 2 "Функция продолжается после цикла" exampleReturn() --> 1 2

Частые ошибки

❌ break вне цикла:
if x > 5 then break -- ОШИБКА! break только внутри циклов end
❌ Думать, что break выходит из if:
for i = 1, 10 do if i == 5 then break -- выходит из for, а не из if end end
❌ Попытка выйти из нескольких циклов сразу:
for i = 1, 10 do for j = 1, 10 do break -- выходит только из внутреннего цикла end end -- Решение: использовать флаг или return
