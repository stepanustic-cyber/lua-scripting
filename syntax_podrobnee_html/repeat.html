🔄 Ключевое слово: repeat

Цикл с постусловием (repeat)

repeat — это ключевое слово, которое начинает цикл с постусловием. Вместе с until оно образует конструкцию, которая гарантированно выполняется хотя бы один раз, проверяя условие выхода после выполнения тела цикла.

📌 Синтаксис:
repeat
    -- тело цикла
until условие
🔄 Отличие от while:
• repeat...until — проверяет условие после выполнения тела
• while...end — проверяет условие до выполнения тела
• repeat...until всегда выполняется минимум 1 раз

Примеры

-- Простейший счётчик local i = 1 repeat print("Итерация " .. i) i = i + 1 until i > 3 -- Вывод: -- Итерация 1 -- Итерация 2 -- Итерация 3
-- Выполнится хотя бы один раз, даже если условие сразу истинно local x = 10 repeat print("Тело цикла выполнилось") until x > 5 --> Тело цикла выполнилось (один раз)
-- Ввод пароля до правильного local password repeat print("Введите пароль:") password = io.read() until password == "secret123" print("Доступ разрешён")
-- Сумма чисел до тех пор, пока пользователь не введёт 0 local sum = 0 local num repeat print("Введите число (0 для выхода):") num = io.read("*n") sum = sum + (num or 0) until num == 0 print("Сумма введённых чисел: " .. sum)
-- Игровой цикл с выходом по команде local score = 0 local command repeat print("Ваш счёт: " .. score) print("Введите команду (add, exit):") command = io.read() if command == "add" then score = score + 10 print("+10 очков!") elseif command == "exit" then print("Игра завершена") else print("Неизвестная команда") end until command == "exit"

Сравнение repeat и while

-- while (проверка ДО) — может не выполниться ни разу local i = 10 while i < 5 do print("while: " .. i) -- не выполнится i = i + 1 end -- repeat...until (проверка ПОСЛЕ) — выполнится хотя бы один раз local j = 10 repeat print("repeat: " .. j) -- выполнится 1 раз j = j + 1 until j > 5

Важные особенности

⚠️ Область видимости переменных:
-- Переменные, объявленные внутри repeat, НЕ видны в условии until repeat local x = 10 print(x) until x > 5 -- ОШИБКА! x здесь недоступна -- Правильно: объявлять переменную ДО цикла local x repeat x = 10 print(x) until x > 5 -- ✅ работает
⚠️ Отсутствие end:
-- В отличие от других циклов, repeat...until НЕ требует end -- while требует end while condition do -- тело end -- ← нужен end -- repeat...until не требует repeat -- тело until condition -- ← нет end

Практические примеры

-- Меню программы local choice repeat print("===== МЕНЮ =====") print("1. Начать игру") print("2. Настройки") print("3. Выход") print("Выберите пункт:") choice = io.read("*n") if choice == 1 then print("Игра началась!") elseif choice == 2 then print("Настройки") elseif choice == 3 then print("До свидания!") else print("Неверный выбор, попробуйте снова") end until choice == 3
-- Игра "Угадай число" math.randomseed(os.time()) local secret = math.random(1, 100) local guess local attempts = 0 repeat print("Угадайте число от 1 до 100:") guess = io.read("*n") attempts = attempts + 1 if guess < secret then print("Больше!") elseif guess > secret then print("Меньше!") end until guess == secret print("Поздравляю! Вы угадали за " .. attempts .. " попыток")

Частые ошибки

❌ Перепутан порядок (как в while):
-- Неправильно (while-синтаксис) repeat i > 10 until print(i) i = i + 1 end -- Правильно repeat print(i) i = i + 1 until i > 10
❌ Бесконечный цикл (условие никогда не становится истинным):
local i = 1 repeat print(i) -- забыли увеличить i until i > 5 --> бесконечный цикл!
💡 Когда использовать repeat...until:
  • Ввод/вывод данных (запрос пароля, валидация ввода)
  • Игровые циклы (до получения команды выхода)
  • Меню и диалоги с пользователем
  • Любая ситуация, когда тело цикла должно выполниться хотя бы один раз
